# Projeto: Clínica IA

## Domínio
Triagem de sintomas de uma clínica

## Integrantes
- Felipe Mello Ramalho Cagnin  
- RM: 569237

## Justificativa
Considero uma área muito interessante para integração de chatbots de IA.

## Dataset
Construído por 10 pares de perguntas e respostas sobre o tema do domínio.

## System Prompt
Defini a língua, a personalidade e acrescentei orientações de como dialogar com humanos.

## Aprendizado
Aprendi como hospedar uma IA, configurá-la, treiná-la e, ao final, interagir com o chatbot criado.
